#Исследовательский анализ поведения пользователей «Яндекс Афиши»

Проект посвящен анализу удержания розничных клиентов платформы за 5 месяцев 2024 года. Основная бизнес-цель исследования — понять, какие группы пользователей с наибольшей вероятностью возвращаются на сервис и делают заказы повторно.

###Основные этапы работы:
* **SQL-выгрузка и предобработка:** Выгрузили массив данных (287 тыс. строк), перевели выручку из тенге в рубли по ежедневному курсу и очистили датасет от b2b-выбросов и технических ботов.
* **Профилирование:** Пересобрали данные на уровне 21.6 тыс. уникальных пользователей, рассчитав параметры их первого заказа и поведенческие метрики.
* **Исследовательский анализ:** Изучили влияние жанров, устройств, регионов, операторов и дней недели на метрику возврата (`is_two`). Проверили продуктовые гипотезы.
* **Корреляционный анализ:** С помощью коэффициента phi_k выявили факторы, наиболее связанные с итоговым количеством покупок.

###технологии:
`Python`, `Pandas`, `SQLAlchemy`, `Phik`, `Seaborn`, `Matplotlib`, `PostgreSQL`
